# Chip Cha Iro: Lost in PawSafe

A single-file HTML FMV arcade game from Reh Dogg Games Plus.

Chip follows a glowing golden treat into PawSafe Forest and must react fast to flashing quick-time prompts to find the way home.

## How to Play

Open `index.html` in a browser, or serve the folder with Live Server.

No npm, build step, or install is required.

## Controls

Keyboard:

- LEFT = `A` or `Arrow Left`
- RIGHT = `D` or `Arrow Right`
- JUMP = `Space` or `Arrow Up`
- RUN = `W` or `R`
- SNIFF = `S`
- BARK = `B`
- Pause = `Escape`

Mobile:

Use the compact on-screen arcade buttons:

- LEFT
- RIGHT
- JUMP
- RUN
- SNIFF
- BARK

## GitHub Pages Upload

Upload the CONTENTS of the release folder directly into the GitHub repository root:

- `index.html`
- `README.md`
- `assets/`

GitHub Pages settings:

- Settings -> Pages
- Build and deployment: `Deploy from branch`
- Branch: `main`
- Folder: `/root`

Do not upload a Vite project, npm files, or source folders. This is a normal static HTML game.

## Ko-fi Product Description

**Chip Cha Iro: Lost in PawSafe** is a cheerful FMV arcade adventure starring Chip, a lovable mascot who follows a glowing golden treat deep into PawSafe Forest. React to flashing quick-time prompts, dodge woodland dangers, cross bridges, sniff out secrets, and help Chip find his way home.

Includes:

- Browser-playable HTML game
- Fullscreen FMV video scenes
- Arcade-style quick-time prompts
- Keyboard and mobile controls
- Score, treats, lives, pause, mute, and volume controls
- GitHub Pages compatible static files
